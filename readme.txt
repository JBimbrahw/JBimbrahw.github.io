To create a website using github pages:
Refer to these refs:
https://www.codingthesmartway.com/how-to-host-a-website-on-github-for-free/
https://docs.github.com/en/pages/getting-started-with-github-pages/configuring-a-publishing-source-for-your-github-pages-site


The 6 steps are:
1) Create Github account
2) Create a repo (named username.github.io)
3) Create a local folder and clone the remote github repository
4) Add your website files to the folder
5) Push the new files to the remote GitHub repository
6) Access the deployed website in the browser

Things to keep in mind:
Make sure to push to the correct branch and set the deployed branch correctly.
The default and deployed branch can be configured in github settings.
